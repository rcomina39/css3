🎨¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para describir la presentación de un documento HTML. Es decir, CSS se encarga de cómo se ven los elementos de una página web.

🧩 ¿Para qué sirve CSS?

Sirve para controlar cómo se ve una página web. Mientras que HTML organiza el contenido (textos, imágenes, botones...), CSS le da el diseño y hace que ese contenido se vea bonito, ordenado y atractivo.

🖼 Ejemplos de lo que puedes hacer con CSS:

  • Cambiar colores de texto o fondo
  • Usar diferentes fuentes y tamaños de letra
  • Crear espacios entre elementos (márgenes, rellenos)
  • Organizar elementos en columnas, filas o cuadrículas
  • Hacer que tu página se vea bien en celulares, tablets y computadoras
  • Crear animaciones y transiciones suaves
  • Ocultar, mostrar o mover cosas en la pantalla
